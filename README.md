# hamedshirzad.com

Personal research website for Hamed Shirzad. Static site served by GitHub Pages.

- `index.html` — the entire site (self-contained: HTML + CSS + a little JS)
- `assets/headshot.jpg` — profile photo (replace this file to swap the photo)
- `assets/Hamed_Shirzad_CV.pdf` — CV linked from the page
- `CNAME` — custom domain configuration (hamedshirzad.com)
